# Secondary 4 Web Authoring Project

This project is a multi-page website developed for the school, EDU4U IGCSE & IAL Centre, as part of a secondary school group assignment introducing students to basic web development concepts for IGCSE ICT module.

The project focuses on creating a structured and interactive website using HTML and CSS.

This repository was created later to archive the project and include it as part of my programming portfolio.
---

## Project Overview

The website consists of several interconnected pages designed to demonstrate fundamental front-end development skills, including layout design, navigation systems, and simple interactivity.

The site contains **6 main webpages**, each serving a specific purpose.

### Webpages

**Home Page**
- Serves as the main landing page
- Introduces the school's programmes and achivements
- Pictures of alumni showcasing as school gallery
- Past exam results carefully researched, calculated and shown in PieCharts
- Former students reviews

**About Us Page**
- Provides a more detailed information about the school including
    - Location
    - Facilities
    - How to apply
    - Latest News

**Sec 2/Sec 3/Sec 4/IAL Page**
- A more detailed information about each classes including subjects offered and class timetables
